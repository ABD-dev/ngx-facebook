LoginOptions

Param Type Details
auth_type (optional) string Optional key, only supports one value: rerequest. Use this when re-requesting a declined permission.
scope (optional) string Comma separated list of extended permissions
return_scopes (optional) boolean When true, the granted scopes will be returned in a comma-separated list.
enable_profile_selector (optional) boolean When true, prompt the user to grant permission for one or more Pages.
profile_selector_ids (optional) string Comma separated list of IDs to display in the profile selector
